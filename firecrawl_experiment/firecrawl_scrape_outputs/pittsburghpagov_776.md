# Occupant Load Placard Permit

A commercial occupant load placard permit is required for the following related to commercial building or structure with an assembly use and/or space:

- Obtain a replacement or new Occupant Load Placard.

Please see [Permit Home](https://www.pittsburghpa.gov/Business-Development/Permits-Licenses-and-Inspections/Permits) for definition of commercial structures.

Over the counter applications will require a [Occupant Load Placard Application(PDF,Â 369KB)](https://www.pittsburghpa.gov/files/assets/city/v/1/pli/documents/commercial_occupant_load_placard.pdf).

## Work Types

- **New Placard:** applies to a placard application associated with a new assembly use and/or space or an increased occupant load for an existing assembly use and/or space.
- **Replacement Placard:** applies to a placard application associated with an existing assembly use and/or space that has not been modified that needs a copy of the placard.
